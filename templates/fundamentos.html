Fundamentos de Programação em Python

1. Estruturas de Seleção

Permitem decisões com if, elif, else.

if x > 10:
    print(\"Maior que 10\")

2. Estruturas de Repetição

Executam blocos várias vezes.

for i in range(5):
    print(i)

3. Vetores e Matrizes

Listas unidimensionais e listas de listas.

lista = [1, 2, 3]
matriz = [[1,2], [3,4]]

4. Funções e Procedimentos

def saudacao():
    print(\"Olá!\")

5. Tratamento de Exceções

try:
    x = int(input(\"Digite um número: \"))
except ValueError:
    print(\"Valor inválido!\")
